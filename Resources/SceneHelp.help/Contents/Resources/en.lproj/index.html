Keyboard Shortcuts

Main workspace shortcuts:

  • Cmd-1 Toggle Binder
  • Cmd-2 Switch to Writing
  • Cmd-3 Switch to Workshop
  • Cmd-4 Toggle Compendium panel
  • Cmd-5 Toggle Summary panel
  • Cmd-6 Toggle Notes panel
  • Cmd-7 Toggle Conversations panel
  • Cmd-8 Toggle Text Generation panel
  • Cmd-K Switch to Text Generation and focus beat input

In generation and workshop inputs, Enter sends and Cmd-Enter inserts a newline.

Text Generation

Use the generation form under the editor to produce prose from a beat or rewrite selected text.

  • When text is selected, the action switches to rewrite mode.
  • Use @ to mention compendium entries and # to mention scenes.
  • The generation form can be collapsed from its lower-right control and restored later.

Rolling Memory

Scene, chapter, and workshop memories keep high-signal state between generations.

  • Workshop memory is chat-session-local and refreshes automatically in background.
  • Scene and chapter memory can be edited from the Summary panel via the book button.
  • Chapter memory supports iterative updates from scene text sources.

Prompt Templates

Prompt templates support variables like {{beat}}, {{context}}, and {{context_rolling}}.

Open Project Settings -> Prompt Templates for variable reference and rendered previews.
